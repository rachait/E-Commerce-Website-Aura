# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume positive intent.
- Provide actionable technical feedback.

## Unacceptable Behavior

- Harassment, hateful language, or personal attacks.
- Trolling, intimidation, or discrimination.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and issues that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to project maintainers.
