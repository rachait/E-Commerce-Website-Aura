## 📝 Getting Started

### 1. Clone & Setup
```bash
cd d:\e\ commerc\ website
cd backend && pip install -r requirements.txt
cd ../frontend && npm install
```

### 2. Start Backend
```bash
cd backend
python seed_db.py  # Create sample data
uvicorn main:app --reload
# Server: http://localhost:8000
# Docs: http://localhost:8000/docs
```

### 3. Start Frontend
```bash
cd frontend
npm run dev
# App: http://localhost:5173
```

### 4. Explore
- Home: See 3D intro scene
- Products: Browse catalog
- Cart: Add items and checkout
- Admin: Manage products (use admin account)
- Chat: Ask the AI assistant
